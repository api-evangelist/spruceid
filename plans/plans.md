# SpruceID Plans

SpruceID is primarily an open-source infrastructure company. Their core libraries
(DIDKit, SSI, SIWE, OID4VCI, OID4VP, isomdl) are available under the Apache-2.0
license at no cost. Commercial offerings are delivered as government and enterprise
identity infrastructure contracts rather than tiered SaaS plans.

## Open Source / Self-Hosted

**Cost:** Free  
**License:** Apache-2.0  
**Access:** GitHub repositories, Rust crates, NPM packages

Includes:
- DIDKit HTTP server (self-hosted)
- SSI core Rust library
- SIWE TypeScript library
- SIWE-OIDC identity provider
- OID4VCI Rust library
- OID4VP Rust library
- isomdl Rust library
- SpruceKit mobile SDKs (iOS/Android)
- SpruceKit Showcase reference wallet app

## Government / Enterprise

**Cost:** Contact for pricing (hello@spruceid.com)  
**Delivery:** Contracts, professional services, implementation support

Includes:
- Identity Gateway (SSO for government applications)
- Digital IDs infrastructure
- Identity Wallet deployment
- Zero Trust Exchange
- Document Intake & Form Modernization
- Legacy system integration
- Security & compliance advisory
- Dedicated implementation support
